Prerequisites:
1. You need Node.js installed to use `npm`
2. You need Python3 installed
3. You have to be able to run the example scene in the `manim_3b1b` folder (see [here](https://github.com/3b1b/manim) and [here](https://github.com/3b1b/manim/pull/2467) for instructions)

Installation

1. Clone the repository
   ```
   git clone git@github.com:Sneezy123/natural-language-to-manim.git
   ```
2. In the directory, run
   ```
   python3 -m 
   ```
3. Navigate to [localhost:8000](http://localhost:8000)
4. In the settings (top right corner), select your AI and input your API key. You can generate a free API key in Google AI Studio. As a model, I recommend using `gemini-3.1-flash-lite`. If you have a Gemini or OpenAI Subscription, choose what you want.
